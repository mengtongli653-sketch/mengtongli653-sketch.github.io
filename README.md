# 随想

有想法就写。这是一个纯档案仓库,不是网站 —— 每篇随笔以 Word 文档(`.docx`)的形式直接存在这里,不用 markdown,也不经过任何构建。

## 用法

- 每篇文章一个 `.docx` 文件,文件名建议用 `年-月-日-标题.docx`,例如 `2026-08-01-周末随想.docx`。
- 想看某篇文章,直接从仓库里下载 / 打开这个文件即可,双击就能用 Word 打开。
- 新增文章:把写好的 `.docx` 放进仓库根目录,`git add`、`commit`、`push` 就行。
